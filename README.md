# Projects 

-[Mobile App Development with iOS] Instaparse Pt.1 (https://github.com/Cecilej05/Cecilej05-Lab2)

This lab involves creating "Instaparse," a photo-sharing app that mimics key functionalities of Instagram and incorporates features from the BeReal app, allowing users to sign up, log in, and post photos from their photo albums, with all user authentication and data managed through a remote custom Parse backend server.

-[Mobile App Development with iOS] Instaparse Pt.2 (https://github.com/Cecilej05/Cecilej05-Lab3/tree/main)

In the second part of the Instaparse project, additional functionality is added to the app, enabling users to capture photos directly with the camera and implement more complex filtering logic for the posts returned in queries.

-[Mobile App Development with iOS] Flashcard (https://github.com/Cecilej05/Cecilej05-Lab4/tree/main)


In this lab, you'll explore SwiftUI's declarative approach by building a flashcard app, focusing on creating a new SwiftUI project in Xcode, understanding the anatomy of a SwiftUI app, designing a basic user interface, modeling data, responding to user input, and implementing gestures and animations.

-[Mobile App Development with iOS] Scavenger Hunt (https://github.com/Cecilej05/Cecilej05-Project1/tree/main)

In this project, you will develop a scavenger hunt app that allows users to attach photos to tasks and displays the locations of those photos on a map using PHPicker to select photos and MapKit to show custom annotations.

-[Mobile App Development with iOS] BeReal Clone Pt.1 (https://github.com/Cecilej05/Cecilej05-Project2/tree/main)

In this project, you'll create the first part of a two-part app that replicates key features of BeReal, enabling users to post photos of their current activities upon receiving random notifications, while integrating the Parse server for login authentication, account creation, and managing photo uploads and retrieval.

-[Mobile App Development with iOS] Memory Game (https://github.com/Cecilej05/Cecilej05-Project4/tree/main)

In this project, you'll build your first SwiftUI application by creating your own version of a Memory Game, focusing on developing UI elements for a single-screen app, updating views based on user input, and implementing game logic to complete the game.
